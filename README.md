# 📱 Contact App - Android Java

A fully functional **Contact Management App** built with Java in Android Studio. This app allows users to store, manage, and interact with their contacts efficiently using SQLite database with complete CRUD operations.

## ✨ Features

- **➕ Add Contact** - Save contact details including name, phone number, email, and profile picture
- **👁️ View Contacts** - Display all contacts in a clean RecyclerView list
- **🔍 Search** - Search contacts by name using SearchView
- **📋 View Details** - Tap any contact to see full details
- **✏️ Edit Contact** - Update existing contact information
- **🗑️ Delete Contact** - Remove unwanted contacts
- **📞 Quick Actions** - Call, message, or email directly from contact details
- **🖼️ Profile Picture** - Add and display contact photos from gallery
- **🌙 Dark/Light Theme** - Switch between light and dark mode
- **🔤 Letter Avatars** - Auto-generated colored avatars for contacts without photos

---

## 🛠️ Tech Stack

- **Language:** Java
- **IDE:** Android Studio
- **Database:** SQLite with custom DBHelper
- **UI Components:** RecyclerView, SearchView, FloatingActionButton, ImageButton
- **Image Handling:** Internal storage with custom URI management
- **Architecture:** MVC Pattern
- **Version Control:** Git & GitHub
